In order to load the bundle

expo start --offline
will work. For running it on an Android device make sure to check if your device is listed in adb available devices by running

adb devices
You can make the connection with your adb device either by connecting it through wire or attaching the bridge wirelessly. Instructions for help!

--offline tag will not work, it will still try to connect to metro server and if your device is not connected to same Wifi it will fail. To run Expo app completely offline (with your phone's wifi off), you need to do the following, Connect your device via USB to your computer. For Android setup ADB and confirm device is connected, (do the same equivalent for iOS). Once the device is connected

Start the expo on localhost using npx expo start --localhost and then select your device and now it should work completely offline
